🌸 Neraki Portfolio

A personal portfolio website showcasing selected front-end and UX/UI projects, with a focus on clarity, usability, and interaction design.

✨ Overview

This portfolio was designed and developed to present my transition into UX/UI and web design, combining my background in animation and visual storytelling with front-end development.

The goal was to create a clean, structured, and approachable experience that reflects both:

technical skills
design thinking
attention to detail
🧠 Key Features
Responsive layout across devices
Clear section-based navigation (Home, About, Skills, Work, Contact)
Project showcase with visual previews and structured descriptions
Interactive hover states and subtle motion
Contact form with Netlify form handling and confirmation page
Accessibility considerations (focus states, reduced motion support)
🛠️ Built With
HTML5
CSS3 (custom properties, responsive layout, animations)
JavaScript (in projects section)
Netlify (deployment & form handling)
Bootstrap Icons
🎯 Design Approach

This project focuses on:

Clarity over complexity
Structured layout and visual hierarchy
User-friendly navigation and flow
Soft motion to support interaction (not distract from it)

The design system is based on reusable variables for:

color
spacing
motion
→ making the interface consistent and scalable
